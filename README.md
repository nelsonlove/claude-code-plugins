# claude-code-plugins

Monorepo of custom Claude Code plugins integrating with macOS tools and local utilities. 17 plugins including Apple Notes, Day One, OmniFocus, Mail, Clawtunes (Apple Music), JD Workflows (Johnny Decimal), Homebrew Audit, TomatoBar, Marked Preview, and more.

Each plugin extends Claude Code's capabilities within the local development environment via MCP servers, hooks, skills, agents, and slash commands.
