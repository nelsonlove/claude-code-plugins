* How to use this file

Toggle states: =S-left= / =S-right= or =C-c C-t=

| Key | State   | Meaning                                |
|-----+---------+----------------------------------------|
| k   | =KEEP=  | Install on next machine                |
| m   | =MAYBE= | Decide later                           |
| d   | =DROP=  | Uninstall now                          |
| p   | =DEP=   | Dependency — managed by Homebrew        |

Other commands:
- =C-c C-x C-c= — Column view (spreadsheet-style overview)
- =C-c \= — Filter by tag (~brew~, ~cask~, ~mas~)
- =C-c / t= — Sparse tree by TODO state
- =C-c C-c= on a =src= block — Execute it

When done triaging, go to [[*Actions]] and run the babel blocks.
