Assemble a daily briefing of what's on the user's plate.

## Your task

1. Query today's events using `pim_query_nodes(type="event")` with date filters
2. Query active tasks in working register, especially flagged or due today
3. Count inbox items in scratch register across all types
4. Present a concise briefing with:
   - Today's calendar
   - Active/due tasks
   - Inbox count
   - Upcoming items (next 2-3 days)
5. If the inbox is large, suggest a triage session
